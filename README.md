# 公視「台語新詞辭庫」複本

來源：<https://www.taigitv.org.tw/taigi-words>

這個複本的目的一方面是做備份，另一方面是讓重複利用變得更容易。

萬分感謝公視台語台對台語的貢獻，與慷慨的開放授權。

## 流程

資料是從[網頁](https://www.taigitv.org.tw/taigi-words?page=1) scrape 而來的。

讀音聲音檔目前沒有 scrape 下來。台羅應該就夠了。
